Hello, welcome to the Reduce Austin Homelessness, website a coordinated solution based effort to address the Homelessness issue in Austin, Texas.

This effort is based on 3 separate but connected (by dataset) mobile apps. The 1st is a Homeless Services Locator. The 2nd isAustin Homeless Activity Reporter. The 3rd is a Austin Homeless Activity Dashboard
